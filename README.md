# Desk Invoice

Modern Billing & Inventory Management Software for Businesses

Desk Invoice is a powerful and modern billing software designed for shops, businesses, freelancers, and retailers. It helps manage invoices, customers, products, payments, and inventory with a clean and user-friendly interface.

Built with modern technologies, Desk Invoice provides fast performance, responsive design, and an easy workflow for daily business operations.

---

## ✨ Features

### 📄 Invoice Management
- Create professional invoices
- Download invoice PDFs
- GST-ready billing system
- Invoice history tracking
- Payment status management

### 👥 Customer Management
- Add and manage customers
- Customer search functionality
- Store customer details
- Walk-in customer support

### 📦 Product & Inventory Management
- Add and update products
- Manage stock quantities
- GST and pricing support
- Low stock alerts

### 💳 Payment Methods
- Cash
- UPI
- Card
- Credit
- Cheque

### 📊 Dashboard & Reports
- Monthly sales analytics
- Revenue tracking
- Outstanding balance tracking
- Top-selling products
- Sales reports

### 🎨 Modern UI/UX
- Fully responsive design
- Smooth animations
- Clean modern interface
- Mobile-friendly layout

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/your-username/desk-invoice.git
cd desk-invoice
```

---

## Install Dependencies

### Client

```bash
cd client
npm install
```

### Server

```bash
cd server
npm install
```

---

# ⚙️ Environment Variables

Create a `.env` file inside the server folder.

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```

---

# ▶️ Run Project

## Start Backend

```bash
cd server
npm run dev
```

## Start Frontend

```bash
cd client
npm run dev
```
